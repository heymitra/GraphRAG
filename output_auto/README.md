# output_auto/

GraphRAG pipeline output for **auto-tuned prompts** runs.

Same Parquet file structure as `output/` (see that README), but produced using prompts
that were automatically generated by `graphrag prompt-tune` based on your corpus.

> **Generated — do not commit.** All files in this directory are regenerated on each indexing run.
> Per-run isolated outputs are stored under `output/runs/{run_id}/`.
