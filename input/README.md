# Input Directory

Place your documents in this directory for GraphRAG processing.

Supported formats:
- `.txt` - Plain text files
- `.pdf` - PDF documents (use extract_pdf.py to convert)

Example:
```
input/
├── document1.txt
├── legal_case.txt
└── research_paper.pdf
```

The files in this directory will be processed by GraphRAG to extract entities, relationships, and communities.
